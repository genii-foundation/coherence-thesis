Providence should expect failure modes not named here. Real participants will discover where design cuts against purpose, where instruments misread, and where power hides.

Correction requires more than openness to feedback. It requires channels through which criticism changes budgets, code, rules, leadership, and, when necessary, the decision to stop. People harmed by the system should not have to become unpaid researchers in order to be believed.

Every serious objection can contribute to better design, but not every objection can be absorbed without changing the thesis. Some evidence may require abandonment of a mechanism or claim. A living process is not one that survives every criticism. It is one capable of giving up what reality has disproved.
